# Discovery — {{title}}

## Problem
What user-visible problem does this change solve? Frame in domain language, not technical terms.

## Bounded context
Which bounded context(s) does this touch? If new, justify why a new context is needed instead of extending an existing one.

## Evidence
- Tickets / support cases / metrics linking to the problem
- Stakeholder requests (with names and dates)

## Constraints
- Regulatory or compliance constraints
- Hard deadlines
- External system contracts that can't change

## Open questions
Things to answer before writing the proposal.
